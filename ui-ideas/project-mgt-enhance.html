C) Installation of the Module - top

  1. Uncompress the Module Folder

    2. Unzip the folder you received after your purchase on CodeCanyon. Once unzipped, the folder will contain the following contents. You will use the upload.zip file to install the module on Perfex CRM.

  2. Upload the Module

    3. Navigate to Setup > Modules in your Perfex CRM dashboard.

    Then select the choose file option to upload a module.

  3. Install the Module

    4. Select the upload.zip file you extracted earlier and click Install.

  4. Activate the Module

    5. Once installed, locate the Project Management Enhancements module in the list of available modules and click Activate to enable the module.

  5. License Activation

    6. Once the module is activated, navigate to the PME License page, enter the requested information along with the purchase key that received upon purchase and click Activate.

    If the license key is valid, you will see a confirmation that the license has been activated:

    After activation, the module will be enabled and ready for use.

  6. Accessing the Module Features

    7. In the subsequent sections we will cover the features that the Project Management Enhancements Module offers, let's explore them one by one.

D) Task Types - top

This module allows you to categorise tasks as general tasks, bugs, or new features—or customise your own. You can add task types directly from the project settings to suit your project needs, with default options like bug, feature, and task.

To see how the task types feature work, go to projects and open any project, then on the menu bar you will see a new menu Settings, click this Settings and you will see Task Types:

Here you will see all your task types.

  1. Adding a New Task Type

    2. To add a new new task type, click the Add Task Type button:

    Provide the name, your desired label colour for this new task type, font colour, and the sort order for the new type of task that you want to create:

    Once you have provided all the information, click Save and you will see that the new task type has appeared with your desired name and label colour:

    You can sort the task types by their name, colour hex code, and the sort order.

    You can also edit or delete a task type using the edit and delete buttons, as shown in the screenshot above.

  2. How to Use a Task Type

    3. When creating a new task, you will see all task types provided by this module, including your custom task types that you created. Open a project and add a new task following the usual process:

    On the new task creation form, provide all other task related detail and for the Task Type, pick any task type, including your custom task types. You will note that all task type titles show in their perspective colours. Then click the save button to create the task:

    On the newly created task you will see the task type label to the left of the task title, in the relevant task type colour:

    On tasks page for any project, you will see all tasks with a task type coloumn. This task type is sortable, meaning that you can sort tasks by a given task type.

    Furthermore, you can search tasks on the tasks page by task a type, simply input your custom task type in the search bar and you will get all tasks with the same task type in the search results.

E) Image/Screenshot Pasting in Task Comments - top

This module allows you to paste Images or screenshots from your clipboard directly into a task. To add an image or screenshot, copy the image or take a screenshot and simply press Ctrl+V on Windows (Command+v on Mac). Then add the comment:

The saved comments will show the picture that you added in the comment:

F) Image/Screenshot Pasting in Task Description - top

You can also paste images or screenshots directly into a task. To add an image or a screenshot in the description of a task, copy the image or take a screenshot, click in the task description area on a task and simply press Ctrl+V on Windows (Command+v on Mac). then click out of the task description:

The saved description will show your pasted picture:

G) Task Info Toggle Feature - top

When you have a thread of comments on a given task, it gives full width of the task modal so you and your team can read the task description and tasks comments clearly. By default the task info will be expanded.

You can minimise/collapse the task info.

H) Task Sharing via Link - top

Sometimes you will need to quickly direct a team member to a task, the click to copy link next to the task name will allow you to do just that.

To share a task with a team member click the task link copy icon next to the task name and share the with your team member:

When your team member pastes this link in their browser or clicks it they can go directly to this task and it will open automatically for them.

You can also copy the task link from the task list page to share a task with your team:

I) Comment Sharing via Link - top

To enhance your work collaboration, you may want to share an idea or an issue you added in a comment. You can do that using the copy link button in comment options menu and send the task link to your team.

When they click this link the browser will automatically scroll to this task comment and the comment in question will have the background highlighted so your team can easily identify the comment you pointed them to.

J) Display Task Label with Task Title on Task List Page - top

This module allows you to sort tasks based on task types. Tasks sorting can be through color or label. You can also copy the task link.

K) Comment Reply in Task Comments - top

This module allows you to reply to a task comment in a threaded conversation. While replying, you can also add images. Furthermore, you can copy the link of any comment in the thread to share with team members the comment will be in focus when they open it through a link.

Once a reply is added to a comment it will show in the thread as child comment.

Below is another example of a threaded conversation on a task comment, where the number of replies in this thread are shown as an expandable option.

When the comment thread is expanded, you will see all comments in this thread, under the the parent comment.

L) Project Owners - top

With the Project Owners feature, designated leaders can access all tasks within their projects without needing to be assigned individually. This streamlines visibility and ensures smoother collaboration while maintaining existing role and permission controls.

You can select project owners while creating or editing a project. Select the users you want to be project owners.

Project owners for each project can be viewed under the project overview tab.

Project owners will be automatically added as task followers for each task within the given project.

M) Loom Pasting into Task description (Beta) - top

Note: This feature is in beta version and may not work on some browsers.

This module allows you paste loom link into task description. When you paste the loom URL for screen recording, the confirmation box will appear.

After you select yes, the video will be embedded if your browser supports it.

N) Multiple Notes on a Project - top

This module allows you to create multiple notes for a project. You can create, edit, delete, pin or unpin the notes.

To create a new note, click on the new note button.

You can also edit or delete any note.

The notes can either be seen in list view or in card view. You can switch to card view by clicking the icon next to new note button.

To mark something as important, you can pin the note, ensuring it remains easily accessible. If the note is no longer needed or relevant, you can unpin it, keeping your workspace organised and clutter-free. This feature helps you prioritise key information, preventing it from getting lost among other notes.

O) Task ID in Task Popup - top

This module displays the Task ID in the task popup, enhancing productivity and improving task management efficiency. The number located to the left of the task name represents the Task ID.
