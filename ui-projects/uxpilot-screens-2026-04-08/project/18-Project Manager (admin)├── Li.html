Access Levels

Admin

Full Access
  • Create, edit, delete projects
  • Manage all access levels
  • Assign users to any role
  • Set project statuses
  • View all project data

Manager

Partial Access
  • Create, edit assigned projects
  • Manage team-level access
  • Assign users to team roles
  • Update project statuses
  • Cannot delete projects

Contributor

Limited Access
  • View assigned projects
  • Edit own assigned tasks
  • Cannot manage access levels
  • Cannot assign users
  • Cannot modify project statuses
