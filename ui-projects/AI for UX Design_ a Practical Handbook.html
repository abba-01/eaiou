Edit (Whole Screen)

The design edit feature, contrary to section edit will re-generate the whole code and it’s great for
Make overall changes
Designing new screens based the selected design
Correcting broken layout
Let’s say I have a design I’m happy with, from here I can “edit” it further and generate the new pages. In this case I want to generate the “Products” page without starting from scratch.
I click on “Edit” (the button on top) and ask UX Pilot to generate the new page.
We’re basically giving it the selected page as context so everything will stay consistent.
result:
You can do the same with other pages by either:
Selecting and editing the newly created page
Or keeping the landing page selected
In this example, the style of the images are off.
If consistent images are crucial for your design, you can edit the screen and ask UX Pilot to improve the generated images.

Use custom assets or a logo

We don’t support direct upload of custom assets or logo, but here’s a simple workaround:
Upload your asset, in this example a logo to https://postimages.org/
Then copy the Direct Link (in this example: https://i.postimg.cc/Wb3336bT/afs-logo.png)
Paste the link in the context and explain what it is.
If you have multiple assets with different sizes, mention their relative size (don’t need to be super precise):
And click generate:
In this particular example it was even able to invert the logo automatically for the footer section. ​
